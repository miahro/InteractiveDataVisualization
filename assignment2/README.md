## Installation

1. Clone repository [Static visualization]()
2. install dependencies:

```bash
poetry install
```

3. Download and copy the dataset to folder:
```bash
/assignment2/data
```


## Running the program

1. To start program run:

```bash
poetry run python3 src/index.py
```
